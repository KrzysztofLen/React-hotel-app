    @mixin navigation() { ... }

    Description

    Helper mixin to style navigation depends what side was chosen.

    Parameters

    None.

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Sizing .nav

    .nav {
  @include navigation(100vh, 94px, hidden, fixed, 0, 0;);
}
